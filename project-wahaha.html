Project Overview 项目概述

PROJECT OVERVIEW

01项目名称

《钙不钙世的，都是小孩》—— 娃哈哈AD钙奶30周年整合营销策划

02项目背景

2026年恰逢娃哈哈AD钙奶上市30周年。面对年轻消费群体不断变化的消费观念与传播环境，品牌需要在延续经典国民记忆的同时，建立与新一代年轻消费者的情感连接。

03项目目标

  • 强化品牌30周年节点传播
  • 激活消费者童年情感记忆
  • 提升年轻群体品牌认知
  • 构建线上线下整合传播体系

04我的职责

  • 品牌策略
  • 用户研究
  • 创意策划
  • 视觉设计
  • 微电影设计、联名设计、快闪店设计、公益活动设计

Project Reflection 项目总结

PROJECT REFLECTION

项目收获

通过本次整合营销策划实践，我从品牌定位、用户洞察到创意策略构建，完整体验了一套品牌传播方案的设计流程。项目不仅关注视觉表现，更关注品牌价值与消费者情感需求之间的连接方式。这一过程让我进一步理解了品牌设计不仅是"设计什么"，更是"为什么这样设计"。
